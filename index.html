Most Recent Project — Self-Hosted Home Lab

This project is a self-hosted Linux homelab designed to explore systems administration, networking, and containerized infrastructure in a real-world environment. The server runs Linux Mint and hosts multiple services using Docker, with Portainer used to manage and monitor containers through a web interface.

Core services include a WireGuard VPN for secure remote access, AdGuard for network-wide DNS filtering and privacy, Uptime Kuma for service monitoring with mobile notifications, and a Jellyfin media server. A centralized homepage dashboard provides quick access to all services while displaying real-time system metrics such as CPU, RAM, and storage usage.

Through this project I gained hands-on experience with Linux server administration, Docker container deployment, VPN networking, and infrastructure monitoring. One of the main challenges was organizing multiple services to run reliably without port conflicts or excessive resource usage, which required configuring Docker networking and carefully managing container resources.

This homelab continues to evolve as a personal environment for experimenting with self-hosted applications, automation, and modern infrastructure tools.
